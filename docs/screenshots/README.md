# Screenshots

Drop the following PNGs in this folder and they'll appear in the root [`README.md`](../../README.md). Recommended width ~1600px (retina), trimmed to the relevant UI.

| File                      | Where it appears                           | What to capture                                                                                                                                                                                                                           |
| ------------------------- | ------------------------------------------ | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `dashboard.png`           | Hero (Screenshots)                         | The full three‑pane dashboard: collections sidebar on the left, a highlight list in the middle, and a highlight's detail pane (note editor + source metadata) on the right. Ideally with a few real highlights and a collection selected. |
| `highlight-toolbar.png`   | Feature → Highlight anything               | A real web page with text selected and the inline Marginalia colour toolbar showing. The colour swatches and note/tag affordance should be visible.                                                                                       |
| `extension-sidepanel.png` | Screenshots + Feature → Review on the page | The extension side panel open next to a page, showing the Highlights / All pages / Collections / Stats tabs.                                                                                                                              |
| `whiteboard.png`          | Feature → Notes & whiteboards              | A tldraw whiteboard (or a rich‑text note) open in the dashboard detail pane.                                                                                                                                                              |

Tips:

- Use a clean browser profile / hide personal bookmarks for a tidy capture.
- Keep the same zoom level across shots so they feel consistent.
- PNG preferred for crisp UI text.
